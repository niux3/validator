update

  • update(field: Field, i?: number, callback?: (field: Field) => void): void

    Updates a field by cleaning it, validating it, and displaying its state. If a callback function is provided, it is executed after the update.

    Parameters

    • field: Field

      The field to be updated.

    • Optionali: number

      An optional index that can be passed to the callback function.

    • Optionalcallback: (field: Field) => void

      An optional callback function to execute after the update. It receives the updated field and the index (if provided) as parameters.

    Returns void

    Example

    // Example usage without a callback
    update(myField);

    // Example usage with a callback
    update(myField, 1, (field, index) => {
      console.log(`Field ${index} updated:`, field);
    });
